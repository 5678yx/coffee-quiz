# 咖啡性格测验 ☕

一个有趣的互动测验，根据你的回答推荐最适合你的咖啡！

## 功能特性

- 🎯 **6个精心设计的问题** - 探索你的性格类型
- 🎨 **生动多彩的设计** - 紫粉橙渐变背景，圆润的 UI
- 📊 **百分比结果** - 查看所有4种性格类型的分布
- ☕ **个性化推荐** - 每种性格都有专属咖啡推荐
- 📱 **完全响应式** - 在手机和桌面设备上完美运行

## 技术栈

- **Next.js 16** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 实用优先的样式
- **Vercel** - 快速部署

## 开始使用

1. 克隆仓库
2. 安装依赖：`npm install`
3. 运行开发服务器：`npm run dev`
4. 打开浏览器访问 `http://localhost:3000`

## 构建生产版本

```bash
npm run build
npm start
```

## 4种性格类型

| 性格 | 咖啡推荐 | 标语 |
|------|-----------|------|
| 🏔️ 勇敢的冒险家 | 双份浓缩咖啡 | 你为激情而活 |
| ☕ 舒适经典 | 中度烘焙滴滤咖啡 | 每一杯都是舒适 |
| 🎉 社交达人 | 卡布奇诺 | 咖啡要和朋友一起分享 |
| 🧘 健康达人 | 燕麦奶美式咖啡 | 每一口都是健康 |
