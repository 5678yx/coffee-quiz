# 咖啡性格测验 - 需求文档

## 项目概述
"你的咖啡性格是什么?"测验 - 一个推荐咖啡的性格测验,结合有趣的互动体验和实用的咖啡推荐。

## 测验问题

### 问题 1: 周末到了,你最想做什么?
- A. 去徒步旅行,挑战自己的极限 → 勇敢的冒险家 🏔️
- B. 在咖啡馆放松,享受宁静时光 → 舒适经典 ☕
- C. 组织朋友聚会,分享快乐时光 → 社交达人 🎉
- D. 瑜伽和健康美食,照顾身体 → 健康达人 🧘

### 问题 2: 你的旅行风格是?
- A. 冒险旅行 - 登山、跳伞、探索未知 → 勇敢的冒险家 🧗
- B. 温馨民宿 - 享受当地的慢生活 → 舒适经典 🏠
- C. 团队旅行 - 和一群朋友一起玩 → 社交达人 👥
- D. 健康度假村 - 瑜伽、水疗、有机食物 → 健康达人 🌿

### 问题 3: 你最喜欢的早餐是?
- A. 快速简单 - 咖啡 + 能量棒,出门! → 勇敢的冒险家 ⚡
- B. 慢慢享用 - 吐司、煎蛋、咖啡 → 舒适经典 🍳
- C. 和朋友一起吃早午餐 - 聊天大笑 → 社交达人 🥂
- D. 健康营养 - 燕麦、水果、绿色果汁 → 健康达人 🥣

### 问题 4: 理想的晚上是?
- A. 参加新活动 - 尝试新运动或课程 → 勇敢的冒险家 🎯
- B. 在家放松 - 好书、好电影、舒适沙发 → 舒适经典 📚
- C. 出门聚会 - 晚餐、酒吧、见朋友 → 社交达人 🌃
- D. 自我关怀 - 冥想、护肤、早睡 → 健康达人 🌙

### 问题 5: 你如何应对压力?
- A. 运动 - 跑步或拳击释放压力 → 勇敢的冒险家 🏃
- B. 安静时光 - 喝咖啡、思考、放慢节奏 → 舒适经典 🎶
- C. 和朋友聊天 - 分享感受、获得支持 → 社交达人 💬
- D. 健康习惯 - 瑜伽、深呼吸、喝茶 → 健康达人 🧘‍♀️

### 问题 6: 如果你能获得一张免费机票,你会去哪?
- A. 冒险目的地 - 新西兰、冰岛、南极洲 → 勇敢的冒险家 ❄️
- B. 温暖舒适的地方 - 意大利托斯卡纳、法国南部 → 舒适经典 🌻
- C. 热闹的城市 - 纽约、东京、巴塞罗那 → 社交达人 🏙️
- D. 健康静修地 - 巴厘岛、哥斯达黎加、印度 → 健康达人 🌴

## 逻辑说明
每个问题的答案映射到一个性格。测验结束时,统计每个性格被选中的次数,并显示百分比结果和所有咖啡推荐。
